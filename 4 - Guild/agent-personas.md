# Guild agent personas (Mapmakers)

Personas in **[1 - People / Personas/](../1%20-%20People/Personas/)** hold **shared** context. Egregore **private** session state stays in `the-mapper-s-ego-memory`. **Export** to a public bundle needs [0 - Housekeeping / Publication / PUBLICATION-AND-RELEASE.md](../0%20-%20Housekeeping/Publication/PUBLICATION-AND-RELEASE.md).

## Egregore vs this tree

- **Egregore** = `people/{github}.md` in private memory, handoffs.  
- **This repo** = guild can share. **Redact** before a publication or vault-snippet push.

## Firefly (context packet)

*Firefly* = `1 - People/Personas/.../index.md` + `3 - Knowledge/patterns/*` + optional `2 - Projects/`. It is not the Egregore log.
