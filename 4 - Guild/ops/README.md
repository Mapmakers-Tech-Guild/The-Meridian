# 4 — Guild / ops

**Guild operations** subfolder: runbooks, agent-facing material, and **personas for agents** (separate from **1 — People**, which is only humans as people).

| Subfolder | Use |
| --- | --- |
| [personas/](./personas/) | Context packs for **guild automations / agents** — *not* “people as people” |
| (add) | SOPs, on-call, bot behavior — as you build them out |
