# Membranes, branches, and what hosts can (and can’t) hide

If a **repository** is **public**, **all branches** on that remote are generally visible — not a “private branch” for secrecy.

| Need | Pattern |
| --- | --- |
| **Drafts private** | Private repository, or no push. |
| **Guild share** | This Mapmakers KB; org controls visibility. |
