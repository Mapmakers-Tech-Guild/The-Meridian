# Guild agent personas (shared knowledge base)

Persona folders here are for **guild-safe** context. **Private** session state stays in Egregore `memory/`. **Publication-bound** copy may get an extra pass per [PUBLICATION-AND-RELEASE.md](../PUBLICATION-AND-RELEASE.md).

## Egregore vs this tree

- **Egregore** = private mind, `people/{github}.md`, handoffs.  
- **This repo** = what the guild is willing to share in the Mapmakers knowledge base. **Redact** before any export to a public or vault-snippets repo.

## Firefly (context packet)

*Firefly* = the bundle for a **guild-oriented** agent: `people/personas/{kebab}/index.md`, `knowledge/patterns/*`, optional `projects/`. It is not the Egregore log.
