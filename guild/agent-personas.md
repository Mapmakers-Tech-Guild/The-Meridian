# Guild agent personas (open knowledge base)

This file lives in the **open** KB, not in Egregore’s private `memory/`. It describes how to use **persona folders** for **public- or community-safe** context. Private session state stays in the Egregore memory repo.

## Egregore vs this doc

- **Egregore** = private shared mind, including tool-driven `people/{github}.md` and handoffs.  
- **This tree** = what you are willing to have in a public or semipublic place. **Redact** before publishing.

## Firefly (context packet)

*Firefly* = the **bundle** you attach so a guild-facing agent is grounded **without** leaking the private memory repo:

1. `people/personas/{kebab}/index.md` (this folder)  
2. Relevant `knowledge/patterns/*` from **this** open KB  
3. A **public** `projects/` blurb if the work is tied to a project  

The **guild bot** (if any) that reads a public git repo is still not the long-term private mind — that is Egregore.
