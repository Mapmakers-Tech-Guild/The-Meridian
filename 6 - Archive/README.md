# 6 — Archive

**Inspiration:** `archive/` trees in work KBs and relay snapshots — **superseded, migrated, or cold** material you still want in git for history, without cluttering **3 - Knowledge** or **5 - Operations**.

| Suggested | Purpose |
| --- | --- |
| `YYYY-MM-topic/` or dated files | Migrated from another system |
| `deprecated/` | Old process docs, replaced by a newer file — link the replacement in a one-line note |

**Rule of thumb:** if it is still the **current** source of truth, it does *not* live in Archive. Move a note here when you **intentionally** retire it.
