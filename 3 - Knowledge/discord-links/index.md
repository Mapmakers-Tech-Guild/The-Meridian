# Discord Links

All URLs ever shared in the guild Discord, exported and organised by domain.

Run the export script to populate this directory:

```bash
DISCORD_TOKEN=<bot-token> DISCORD_GUILD_ID=<guild-id> \
  npm run export:discord-links
```

See `scripts/export-discord-links.mjs` for full documentation.
